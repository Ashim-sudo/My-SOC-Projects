# 18 - Detecting Web Attacks

## What I Learned
- Common web attacks (SQL Injection, XSS, Directory Traversal, Brute Force, Command Injection, etc.)
- How to detect web attacks using logs and network traffic
- Analyzing web server logs (Apache/Nginx) for malicious patterns
- Identifying suspicious HTTP requests, user agents, and payloads
- Using SIEM and Wireshark for web attack detection

## Key Takeaways
- Web applications are one of the most targeted vectors by attackers
- Learned practical detection methods for OWASP Top 10 attacks
- Understood the importance of proper web logging and monitoring

## Concepts Covered
- OWASP Top 10 Web Attacks
- Web Log Analysis
- HTTP Traffic Inspection
- Malicious Payload Detection
