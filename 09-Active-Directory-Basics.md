# 09 - Active Directory Basics

## What I Learned
- What Active Directory (AD) is and why companies use it
- Key AD components:
  - Domain Controller
  - Users and Groups
  - Organizational Units (OUs)
  - Domain, Forest, and Trusts
- How authentication works in Windows environments (Kerberos, NTLM)
- Common AD objects and hierarchy

## Key Takeaways
- Understood that most companies run on Active Directory
- Learned how users, computers, and permissions are managed
- Realized why AD is a high-value target for attackers

## Concepts Covered
- Active Directory Domain Services (AD DS)
- Authentication Protocols
- Group Policy
- AD Hierarchy
