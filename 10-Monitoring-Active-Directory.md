# 10 - Monitoring Active Directory

## What I Learned
- How to monitor Active Directory for suspicious activities
- Important Event IDs in AD security logs
- Detecting common AD attacks:
  - Privilege escalation
  - Suspicious user/group changes
  - Lateral movement
  - Kerberoasting
- Using SIEM or Event Viewer to find anomalies in high-volume AD logs

## Key Takeaways
- Active Directory is one of the most critical assets to monitor in a company
- Learned which events indicate potential compromise
- Understood the importance of monitoring user account changes and group membership

## Important Event IDs
- 4728 / 4729 → User added/removed from security group
- 4732 / 4733 → User added/removed from distribution group
- 4624 / 4625 → Logon events
- 4672 → Special privileges assigned

## Tools / Concepts Covered
- Active Directory Monitoring
- Security Event Logs
- Anomaly Detection in AD
- Common AD Attack Techniques
